# BankNode Authentication Model Deployment (FastAPI)

This project uses **FastAPI** to deploy a **BankNode Authentication Machine Learning Model** as an API.  
The application allows clients to send input data and receive authentication predictions from the ML model in real time.

## Features
- FastAPI backend
- ML model loaded using Python
- Prediction endpoint for authentication
- Easy to deploy on Render

## Endpoints
- `GET /` – Check if the API is running
- `POST /predict` – Send input data to get ML prediction
